# Archived R Utilities

**Archived:** February 11, 2026

R validation and extraction scripts. These were used during simulation development and debugging.

- `check_actions.R` - Action export validation
- `check_actors.R` - Actor system validation
- `check_history.R` - Historical data validation
- `check_metrics.R` - Metrics validation
- `extract_ground_truth_from_rds.R` - Ground truth extraction from .rds files
- `extract_rds_simple.R` - Simplified extraction script
- `generate_prompts_v2.R` - Old prompt generation (superseded by Python prompt system)
