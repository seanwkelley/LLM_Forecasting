# Archived Analysis Scripts

**Archived:** February 11, 2026

One-off analysis and utility scripts from the forecasting research. These were used for specific investigations and are not part of the main pipeline.

## Power Analysis & Statistical Design
- `agent_count_tradeoff.py` - N agents x K scenarios trade-off analysis
- `effect_size_from_real_data.py` - Cohen's d estimation from 5-period data
- `power_analysis.py` - Power analysis for paired t-test across scenarios

## Multi-Scenario Analysis
- `analyze_scenario_variation.py` - Collapse probability / action variation across scenarios
- `analyze_discussion_diversity.py` - Agent debate transcript diversity
- `compare_agent_statements.py` - Cross-scenario content comparison
- `generate_mock_multiscenario_data.py` - Synthetic data for pipeline testing

## Persona & Prompt Analysis
- `analyze_condition3_results.py` - Personalized condition performance
- `compare_original_vs_enhanced.py` - Original vs enhanced prompt comparison
- `compare_prompt_vs_groundtruth.py` - Prompt-based vs ground truth
- `create_enhanced_prompts.py` - Enhanced prompt generation
- `investigate_reasoning.py` - Agent reasoning pattern analysis
- `test_with_full_information.py` - Full information baseline test
