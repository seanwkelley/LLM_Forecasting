## Key Improvements in V2

1. **Faction Names Throughout**
   - "Novaris" and "Tethys" replace generic "Major power" / "Smaller power"
   - Applied to events, descriptions, and all text

2. **Incremental Updates (Not Cumulative)**
   - Periods 2+ show only changes since last period
   - Prevents exponential bloat (Period 10 is ~100 lines, not 500+)

3. **Natural Language Descriptions**
   - "Strong and growing international support" not "76%"
   - "Limited targeted sanctions" not "25% implementation"

4. **Professional Prose Style (Economist/FT)**
   - Connected paragraphs, not disconnected sections
   - No boilerplate filler phrases
   - Narrative event bullets (no "Category: description" format)

5. **Individual Period Files**
   - Each period is standalone .txt file
   - Easy distribution to participants
   - instructions.txt provides study overview

## V1 → V2 Changes

### Removed
- Cumulative history (all past events/actions)
- "Consider factors such as:" prescriptive section
- Raw percentages in text
- "Simulated" prefix on events
- Generic faction references in event descriptions
- Boilerplate like "Economic warfare continues to intensify" (every period)
- Verbose actions preamble
- Dashboard-style changes ("Moved from X to Y")

### Added
- Brief constant scenario context (2-3 sentences)
- "Developments Since Last Period" section
- Prose-style changes descriptions
- Connected current situation paragraphs
- Clean narrative event bullets
- Individual period files with instructions

### Improved
- Event processing (faction name substitution)
- Military balance descriptions (Novaris/Tethys not Defender/Aggressor)
- Changes section (natural prose not metrics)
- Current situation (flowing paragraphs not templates)

## Why These Changes Matter

### For Participants
- Easier to read (professional style, no jargon)
- Manageable length (no bloat in later periods)
- Clear trajectory (changes section shows direction)
- Reduced cognitive load (focus on "what's new")

### For Researchers
- More realistic (mimics real intelligence briefings)
- Better engagement (participants stay motivated)
- Cleaner data (less confusion about metrics)
- Easier distribution (individual files)
